## 📌 About the Project

This project was developed as a practical assessment for the **UC00617 - Using Git and GitHub Services** course unit of the **CET TPSI** programme at **ATEC — Training Academy**.

The main goal is not the web project itself, but the correct demonstration of Git and GitHub best practices throughout its development — from repository initialization to collaborative work with conflict simulation.

---

## ✨ Git Concepts Demonstrated

| Concept | Description |
|---|---|
| 🗂️ Initialization | Local repository creation with `git init` and initial configuration |
| 📝 Commit Management | Incremental commits with messages following Conventional Commits |
| 🔍 Analysis & History | Usage of `git status`, `git diff`, `git log` and `git log --oneline` |
| 🌿 Branches & Merge | Feature branches creation and integration into `main` |
| 🏷️ Tags & Versioning | Lightweight and annotated tags with semantic versioning |
| ⚔️ Conflict Simulation | Intentional conflict between branches and manual resolution |
| 🤝 Collaborative Work | Shared repository with teamwork on GitHub |

---

## 🛠️ Technologies Used

- **HTML5** — semantic page structure
- **CSS3** — styling and layout
- **JavaScript** — page interactivity
- **Git** — local version control
- **GitHub** — remote repository and collaborative work
